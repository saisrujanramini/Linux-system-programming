# Linux System Programming

A structured collection of Linux System Programming programs, concepts, architecture notes, and presentation materials developed during learning and implementation.

---

## Overview

This repository contains practical implementations and study materials related to Linux System Programming.

The goal of this repository is to build a strong understanding of how Linux works internally through:

* System-level programming
* Process management
* File operations
* Inter Process Communication (IPC)
* Signals
* Process architecture
* Linux APIs
* Hands-on coding examples
* Topic-wise presentations and documentation

This repository serves as both:

* Learning documentation
* Practical implementation reference
* Revision material for Linux internals
